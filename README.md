# 🌍 여행지 도시 돌림판

대륙별로 여행지를 선택하고 돌림판을 돌려 랜덤하게 여행지를 결정하는 웹 애플리케이션입니다.

## 🚀 기능

- **대륙별 여행지 선택**: 아시아, 유럽, 북미, 오세아니아 중 선택
- **도시 추가/삭제**: 원하는 도시를 추가하거나 삭제 가능
- **돌림판 애니메이션**: 부드러운 애니메이션과 함께 랜덤 선택
- **결과 팝업**: 당첨된 도시를 팝업으로 표시

## 📦 사용 방법

1. 대륙을 선택합니다 (아시아, 유럽, 북미, 오세아니아)
2. 원하는 도시를 추가할 수 있습니다
3. "돌리기" 버튼을 클릭하여 랜덤 여행지를 선택합니다

## 🛠️ 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API

## 📝 배포

GitHub Actions를 통해 자동으로 GitHub Pages에 배포됩니다.

### 배포 설정

1. GitHub 저장소의 Settings > Pages로 이동
2. Source를 "GitHub Actions"로 선택
3. `main` 또는 `master` 브랜치에 푸시하면 자동으로 배포됩니다
